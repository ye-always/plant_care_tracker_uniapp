# 静态资源说明

## 图标资源要求

### TabBar 图标
位置：`static/tabbar/`

需要的图标文件：
- `plant.png` - 未选中的植物图标（建议尺寸：40x40px，格式：PNG）
- `plant-active.png` - 选中的植物图标（建议尺寸：40x40px，格式：PNG）

### App 图标
位置：`static/icon/`

需要的图标文件：
- `icon.png` - App图标（建议尺寸：1024x1024px，格式：PNG）
- `icon-foreground.png` - Android自适应图标前景（建议尺寸：512x512px）
- `icon-background.png` - Android自适应图标背景（建议尺寸：512x512px）

### 启动页
位置：`static/splash/`

需要的文件：
- `splash.png` - 启动页图片（建议尺寸：1080x1920px，格式：PNG）

## 图标设计建议

### 颜色方案
- 主色调：#10b981（绿色）
- 辅助色：#6b7280（灰色）
- 背景色：#ffffff（白色）

### TabBar 图标
- 未选中状态：灰色 (#6b7280)
- 选中状态：绿色 (#10b981)
- 风格：简约线条或填充风格

### App 图标
- 主题：植物相关（叶子、花朵等）
- 风格：现代简约
- 背景色：建议使用绿色或白色

## 如何添加图标

1. 准备符合要求的图标文件
2. 将图标文件放置到对应的目录中
3. 确保文件名与上述要求一致
4. 重新构建应用

## 在线工具推荐

- 图标设计：Figma、Sketch、Adobe XD
- 图标库：Iconfont、Flaticon、Material Icons
- 图片压缩：TinyPNG、Squoosh

## 注意事项

- 所有图标文件必须使用PNG格式
- 确保图标在不同设备上清晰可见
- 遵循各平台的设计规范
- 图标文件大小建议控制在100KB以内
