# 植物养护记录系统 - 功能结构文档

## 项目概述

**项目名称**: Plant Care Tracker (植物养护记录)
**技术栈**: Vue 3 + TypeScript + uni-app + Tailwind CSS
**项目类型**: 跨平台移动应用（支持H5、微信小程序、支付宝小程序等多端）

## 项目结构

```
plant_care_tracker_uniapp/
├── src/
│   ├── components/          # 组件目录
│   ├── hooks/              # 自定义Hooks
│   ├── types/              # TypeScript类型定义
│   ├── styles/             # 样式文件
│   ├── App.vue             # 根组件
│   └── main.ts             # 应用入口
├── dist/                   # 构建输出目录

├── package.json            # 项目配置
├── vite.config.ts          # Vite配置
├── tailwind.config.js      # Tailwind CSS配置
└── tsconfig.json           # TypeScript配置
```

## 核心功能模块

### 1. 植物管理模块

#### 1.1 添加植物
**组件**: [AddPlantDialog.vue](plant_care_tracker_uniapp/src/components/AddPlantDialog.vue)

**功能描述**:
- 支持添加新植物记录
- 表单字段：
  - 植物名称（必填）
  - 品种（可选）
  - 种植日期
  - 位置（可选）
  - 标签（支持多个，逗号分隔）
  - 浇水间隔（天数）
  - 施肥间隔（天数）
  - 备注（可选）

**技术实现**:
- 使用uni-popup组件实现弹窗
- 使用uni-forms实现表单验证
- 使用uni-datetime-picker选择日期
- 标签输入支持逗号分隔解析

#### 1.2 植物展示
**组件**: 
- [PlantCard.vue](plant_care_tracker_uniapp/src/components/PlantCard.vue) - 网格卡片视图
- [PlantListItem.vue](plant_care_tracker_uniapp/src/components/PlantListItem.vue) - 列表视图

**功能描述**:
- 支持网格和列表两种展示模式
- 显示植物基本信息：
  - 植物图片（支持占位符）
  - 植物名称和品种
  - 种植日期
  - 标签展示
  - 最近养护记录
- 支持点击查看详情和删除操作

#### 1.3 植物详情
**组件**: [PlantDetailDialog.vue](plant_care_tracker_uniapp/src/components/PlantDetailDialog.vue)

**功能描述**:
- 展示植物完整信息
- 显示养护记录历史
- 支持添加养护记录
- 支持删除养护记录
- 养护类型：浇水、施肥、修剪、病害处理、虫害处理

### 2. 养护记录模块

#### 2.1 养护记录管理
**功能描述**:
- 为每株植物添加养护记录
- 记录类型：
  - 浇水
  - 施肥
  - 修剪
  - 病害处理
  - 虫害处理
- 每条记录包含：
  - 养护类型
  - 养护日期
  - 备注（可选）

#### 2.2 快速浇水功能
**组件**: [QuickWaterDialog.vue](plant_care_tracker_uniapp/src/components/QuickWaterDialog.vue)

**功能描述**:
- 批量选择多株植物进行浇水记录
- 使用uni-checkbox实现多选
- 一键为选中植物添加浇水记录
- 自动记录当前日期

### 3. 搜索与筛选模块

**组件**: [FilterBar.vue](plant_care_tracker_uniapp/src/components/FilterBar.vue)

**功能描述**:
- **搜索功能**: 按植物名称或品种搜索
- **排序方式**:
  - 按名称排序
  - 按种植日期排序
  - 按最近养护时间排序
- **位置筛选**: 按植物位置筛选
- **标签筛选**: 按植物标签筛选
- **视图切换**: 网格视图/列表视图切换

**技术实现**:
- 使用uni-search-bar实现搜索
- 使用uni-data-select实现下拉选择
- 响应式布局，支持移动端和桌面端

### 4. 统计分析模块

**组件**: [StatsDashboard.vue](plant_care_tracker_uniapp/src/components/StatsDashboard.vue)

**功能描述**:
- **统计数据卡片**:
  - 总植物数量
  - 浇水次数统计
  - 施肥次数统计
  - 修剪次数统计
- **最近养护活动**: 显示最近10条养护记录
- **位置分布**: 统计各位置的植物数量

### 5. 数据持久化模块

**Hook**: [usePlants.ts](plant_care_tracker_uniapp/src/hooks/usePlants.ts)

**功能描述**:
- 使用uni-app的storage API实现数据持久化
- 数据存储键：
  - `plant-tracker-plants`: 植物数据
  - `plant-tracker-care-records`: 养护记录
  - `plant-tracker-health-issues`: 健康问题记录
- 自动同步数据到本地存储
- 支持CRUD操作：
  - 添加/更新/删除植物
  - 添加/删除养护记录
  - 添加/更新/删除健康问题

## 数据模型

### Plant (植物)
```typescript
interface Plant {
  id: string;                    // 植物ID
  name: string;                  // 植物名称
  species?: string;              // 品种
  plantedDate: string;           // 种植日期
  imageUrl?: string;              // 图片URL
  notes?: string;                // 备注
  location?: string;             // 位置
  tags?: string[];               // 标签数组
  wateringInterval?: number;     // 浇水间隔（天）
  fertilizingInterval?: number;  // 施肥间隔（天）
}
```

### CareRecord (养护记录)
```typescript
interface CareRecord {
  id: string;                    // 记录ID
  plantId: string;               // 植物ID
  type: 'water' | 'fertilize' | 'prune' | 'disease' | 'pest';
  date: string;                  // 养护日期
  notes?: string;                // 备注
}
```

### HealthIssue (健康问题)
```typescript
interface HealthIssue {
  id: string;                    // 问题ID
  plantId: string;               // 植物ID
  type: 'disease' | 'pest';      // 问题类型
  description: string;           // 描述
  date: string;                  // 发现日期
  resolved: boolean;             // 是否已解决
  resolvedDate?: string;         // 解决日期
}
```

## 用户界面设计

### 主界面布局
- **顶部导航栏**: 应用标题和快捷操作按钮
- **标签页切换**: 我的植物 / 统计分析

- 支持移动端优先设计
- 响应式网格布局：
  - 小屏幕: 1列
  - 中等屏幕: 2列
  - 大屏幕: 3-4列
- 自适应字体大小和间距

## 技术特性

### 1. 跨平台支持
- 支持H5、微信小程序、支付宝小程序、百度小程序、字节跳动小程序、QQ小程序等多端部署
- 使用uni-app框架实现一套代码多端运行

### 2. 状态管理
- 使用Vue 3 Composition API
- 自定义usePlants Hook管理植物和养护记录状态
- 响应式数据绑定和自动更新

### 3. 表单验证
- 使用uni-forms组件实现表单验证
- 支持必填字段验证和自定义验证规则

### 4. 数据持久化
- 使用uni-app storage API
- 自动同步数据变更到本地存储
- 支持数据导入导出（预留接口）

### 5. 日期处理
- 使用date-fns库处理日期格式化和计算
- 支持日期选择器和日期格式化显示
