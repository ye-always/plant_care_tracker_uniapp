# 植物养护记录系统 - Android/iOS 打包指南

## 概述

本文档详细说明如何将植物养护记录系统打包为Android和iOS移动应用。项目已基于uni-app框架进行兼容性改造，支持多平台打包。

## 环境准备

### 必需工具

1. **Node.js** (v16.x 或更高版本)
   ```bash
   node --version
   npm --version
   ```

2. **HBuilderX** (推荐) 或 **uni-app CLI**
   - HBuilderX下载：https://www.dcloud.io/hbuilderx.html
   - 选择"App开发版"

3. **Android Studio** (用于Android打包)
   - 下载：https://developer.android.com/studio
   - 需要配置Android SDK

4. **Xcode** (用于iOS打包，仅限macOS)
   - 从App Store安装
   - 需要Apple开发者账号

### 开发环境配置

```bash
# 安装项目依赖
npm install

# 安装uni-app CLI (如果使用CLI方式)
npm install -g @dcloudio/uniapp-cli
```

## 打包流程

### 方式一：使用HBuilderX打包（推荐）

#### Android打包步骤

1. **导入项目**
   - 打开HBuilderX
   - 文件 → 导入 → 从本地目录导入
   - 选择项目根目录

2. **配置应用信息**
   - 打开 `src/manifest.json`
   - 修改应用名称、版本号等信息
   - 配置应用图标和启动页

3. **Android打包**
   - 发行 → 原生App-云打包
   - 选择Android平台
   - 配置签名信息（首次打包需要）
   - 点击打包

4. **获取APK**
   - 打包完成后下载APK文件
   - 可直接安装到Android设备

#### iOS打包步骤

1. **导入项目**（同Android）

2. **配置应用信息**
   - 打开 `src/manifest.json`
   - 修改iOS相关配置
   - 配置Bundle ID、证书等

3. **iOS打包**
   - 发行 → 原生App-云打包
   - 选择iOS平台
   - 上传iOS证书和描述文件
   - 点击打包

4. **获取IPA**
   - 打包完成后下载IPA文件
   - 使用Xcode或TestFlight安装

### 方式二：使用CLI打包（仅适用于H5和小程序）

**重要说明**：uni-app的原生App打包（Android APK/iOS IPA）必须使用HBuilderX或uni-app官方CLI工具，无法通过简单的vite build命令完成。

#### H5打包

```bash
# 构建H5应用
npm run build:h5

# 构建产物位置
# dist/ 目录，包含index.html和资源文件
```

#### 小程序打包

```bash
# 微信小程序
npm run build:mp-weixin

# 支付宝小程序
npm run build:mp-alipay

# 百度小程序
npm run build:mp-baidu

# 字节跳动小程序
npm run build:mp-toutiao

# QQ小程序
npm run build:mp-qq

# 构建产物位置
# dist/ 目录，包含小程序代码
```

#### App打包（需要HBuilderX）

```bash
# 构建App资源（仅生成资源，不生成APK/IPA）
npm run build:app

# 注意：此命令只生成资源文件，需要使用HBuilderX进行最终打包
```

## 应用配置

### manifest.json 配置说明

```json
{
  "name": "植物养护记录",           // 应用名称
  "appid": "__UNI__PLANTCARE",     // 应用ID（需在DCloud平台注册）
  "versionName": "1.0.0",          // 版本名称
  "versionCode": "100",            // 版本号
  "app-plus": {
    "distribute": {
      "android": {
        "permissions": [            // Android权限配置
          "<uses-permission android:name=\"android.permission.CAMERA\"/>",
          "<uses-permission android:name=\"android.permission.READ_EXTERNAL_STORAGE\"/>"
        ]
      },
      "ios": {
        "idfa": false              // iOS广告标识符
      }
    }
  }
}
```

### 获取AppID

1. 访问DCloud开发者中心：https://dev.dcloud.net.cn/
2. 注册/登录账号
3. 创建应用获取AppID
4. 将AppID填入manifest.json

## 图标和启动页配置

### 图标要求

- **App图标**: 1024x1024px PNG格式
- **TabBar图标**: 40x40px PNG格式
- **启动页**: 1080x1920px PNG格式

### 配置步骤

1. 准备图标文件
2. 放置到对应目录：
   - App图标：`static/icon/icon.png`
   - TabBar图标：`static/tabbar/plant.png`
3. 在HBuilderX中打开manifest.json
4. 点击"图标配置"上传图标
5. 自动生成各尺寸图标

## 签名配置

### Android签名

1. **生成签名文件**
   ```bash
   keytool -genkey -alias plantcare -keyalg RSA -keysize 2048 -validity 10000 -keystore plantcare.keystore
   ```

2. **配置签名**
   - 在HBuilderX打包界面选择"使用自有证书"
   - 上传keystore文件
   - 输入密码和别名

### iOS签名

1. **获取证书**
   - 登录Apple Developer账号
   - 创建App ID
   - 生成开发/发布证书
   - 创建描述文件

2. **配置签名**
   - 在HBuilderX打包界面上传证书
   - 选择对应的描述文件

## 常见问题

### 构建失败

1. **依赖问题**
   ```bash
   # 清除缓存重新安装
   rm -rf node_modules
   npm install
   ```

2. **配置问题**
   - 检查manifest.json配置是否正确
   - 确认AppID是否有效

3. **环境问题**
   - 检查Node.js版本
   - 确认HBuilderX版本

### 运行时错误

1. **白屏问题**
   - 检查main.ts导出是否正确
   - 确认页面路径配置

2. **样式问题**
   - 检查Tailwind CSS配置
   - 确认样式文件引入

3. **组件问题**
   - 检查组件注册
   - 确认easycom配置

### 打包问题

1. **签名失败**
   - 检查证书是否过期
   - 确认密码正确

2. **图标问题**
   - 检查图标尺寸
   - 确认文件格式

3. **权限问题**
   - 检查权限配置
   - 确认用户授权
